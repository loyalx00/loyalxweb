## 差し替え手順
1. 上記ファイル名でこのフォルダに画像を置く（ファイル名は必ず合わせる）
2. ブラウザのキャッシュをクリア (Cmd+Shift+R / Ctrl+Shift+R)
3. ページをリロードして確認

## フォールバック順
